# Controle de Despensa (Simples)

Uma aplicação feita em **Python Puro**, utilizando apenas arquivos `.txt` para ajudar a controlar a validade dos alimentos e evitar o lixo orgânico.

## Como testar no seu computador

1. Faça o clone do projeto:
   `git clone https://github.com/MiguelMarqz/despesa-consciente.git`
2. Instale a ferramenta de teste:
   `pip install -r requirements.txt`
3. Rode o programa:
   `python despensa.py`

## 📚 Requisitos Atendidos neste Portfólio
* **Desenvolvimento da Aplicação:** Interface CLI simples com `while` e `input`.
* **Dor Real:** Redução do desperdício de alimentos.
* **Documentação:** Este README.
* **Dependências:** Declaração simples no `requirements.txt`.
* **Testes:** Configurados com `pytest`.
* **Análise Estática:** Uso do módulo nativo `py_compile` para validação de sintaxe sem dependências extras.
* **CI (Continuous Integration):** Pipeline configurada via GitHub Actions.
* **Versionamento Semântico:** Releases criadas através de Git Tags (Ex: v1.0.0).
